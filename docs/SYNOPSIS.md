## 1. Abstract
The **Destiny Tour & Travels** is a modern, high-performance travel booking and AI-powered planning application designed for tourists exploring Himachal Pradesh. This project is a comprehensive web-based platform built with **Next.js 15**, **Firebase**, and **Genkit AI** (powered by Google Gemini & Groq). It features an immersive "Visual Overdose" UI that showcases the beauty of the mountains through interactive elements and smooth animations. The platform includes an intelligent **AI Trip Planner** that crafts personalized itineraries, interactive maps for real-time route calculation, and a premium fleet showcase for vehicle bookings. This project demonstrates the potential of AI-driven travel solutions and modern web frameworks to create a unique, user-centric travel experience.

## 2. Name of the Project
Destiny Tour & Travels (Himachal Perspective)

## 3. Objectives
- To promote tourism in Himachal Pradesh by providing a visually stunning and easy-to-use digital platform.
- To utilize **Generative AI** for creating personalized, intelligent trip itineraries based on user preferences.
- To implement **interactive maps** for real-time distance calculation and fare estimation for tourists.
- To offer a seamless user interface with **premium animations** using Framer Motion and Shadcn UI.
- To simplify the travel inquiry and booking process through direct WhatsApp integration.
- To provide a secure and reliable backend using **Firebase Firestore** for managing user data and trip records.

## 4. Users Views
- **Admin/Owner**: Manages destinations, updates vehicle fleet, and monitors user inquiries.
- **User/Tourist**: Explores Himachal, plans AI-driven trips, calculates travel costs, and initiates bookings via WhatsApp.
